Formatting Link List Item Section ( Sample Page using this Formatting. ) : Steps

Link List Item section for each list article needs to be formatted using following steps

  • Enter the List Title in First text field marked "Enter List Item Title".
  • Enter the URL in Second text field marked "Enter URL".
  • Enter the List Item content in the field marked "Enter Paragraph content here"
  • Click "Create HTML" Button.
  • The HTML will be created on right side text field.
  • Copy the HTML text and paste it in the document.
  • Refresh the page to make sure all content is clear.
  • Repeat the steps from step 2 to step 7 for each link list item.
